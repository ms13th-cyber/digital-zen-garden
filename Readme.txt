ダッシュボードを「静寂の庭」へ。ノイズを徹底排除し、隠しボタンで俗世（メニュー）と繋がるミニマリズム・ツール。

== Description ==

Digital Zen Garden は、WordPress管理画面の入り口であるダッシュボードから、一切の情報を削ぎ落とすためのプラグインです。

ログインした瞬間に飛び込んでくる大量の通知やウィジェットは、時に集中力を削ぐノイズとなります。このプラグインはそれらを浄化し、砂紋が広がる静かな空間を提供します。

主な特徴：

* **ダッシュボードの浄化**: ウィジェット、タイトル、通知、フッターテキストをすべて排除。
* **隠しスイッチ（庭石）**: 右下の小さなアイコンをクリックした時だけ、メニューや管理バーを表示。
* **二面性の両立**: 適用をダッシュボードのみに制限することで、静寂と実務の利便性を両立。
* **超軽量構成**: 1つのPHPファイルのみで動作し、外部リソースを一切消費しません。

== Installation ==

1. `digital-zen-garden.php` を `/wp-content/plugins/` にアップロードします。
2. プラグインを有効化します。
3. ダッシュボードへ移動し、右下の「石」をクリックして動作を確認してください。
