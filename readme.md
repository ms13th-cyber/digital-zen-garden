# Digital Zen Garden

A minimalist WordPress plugin that transforms your cluttered dashboard into a "Silent Garden." Focus on your thoughts by stripping away every visual noise with a single click.

---

## Key Features

- **The Purified Dashboard**: Completely removes all dashboard widgets, "Dashboard" titles, and annoying notices. Only a subtle "Samon" (ripple) pattern remains.
- **Stealth Interaction (The Zen Stone)**: All administration menus and the top bar are hidden by default. Click the small "Garden Stone" at the bottom right to reveal them.
- **Deep Cleansing**: Automatically silences footer credits ("Thank you for creating with WordPress") and version info to maintain absolute stillness.
- **Optimized for Focus**: Only affects the main Dashboard (index.php), ensuring your actual administrative tasks in other pages remain standard and functional.
- **Zero-Bloat Design**: A single-file architecture with no external dependencies or assets. Just pure, sharpened code.

## Installation

1. Upload the `digital-zen-garden.php` file directly to your `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Visit your Dashboard and find the small stone to start your meditation.

## 主な機能（日本語）

WordPressのダッシュボードを「静寂の庭」へと変えるミニマリズム・プラグイン。煩雑な通知やウィジェットを一掃し、思考のための空間を提供します。

- **ダッシュボードの浄化**: ウィジェット、タイトル、通知、案内テキストをすべて排除。残るのは目に優しい「砂紋」の背景のみです。
- **隠しスイッチ「庭石」**: 標準のメニューや管理バーは通常、気配を消して隠れています。右下の小さな「庭石」をクリックした時だけ、俗世の機能（メニュー）がスッと現れます。
- **徹底したノイズ除去**: フッターのクレジットやバージョン情報までサーバーサイドで抹消。視界に入るすべての定型文を浄化します。
- **実用的な静寂**: 適用範囲をダッシュボードのみに限定。投稿や設定などの実務ページは標準のUIを維持するため、作業効率を損ないません。
- **究極の単一ファイル**: 外部ファイルを一切読み込まず、1つのPHPファイル内でロジックとデザインを完結。ms13th-cyberの哲学である「軽量・単一機能」を具現化しました。
